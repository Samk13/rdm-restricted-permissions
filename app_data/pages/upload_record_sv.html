Detta dokument förklarar processen för att ladda upp en post.

För att starta processen, klicka på knappen "Snabbskapa" och klicka på "Ny uppladdning".

Efter att ha klickat på "Ny uppladdning" kommer du att dirigeras till uppladdningssidan.

På uppladdningssidan, klicka på "Välj en Community".

Efter att ha klickat på "Välj en Community" kommer en ny modal att visas. Klicka på önskad community.

Du kommer att se din valda community på uppladdningssidan.

För att ladda upp dina filer, klicka på knappen "Ladda upp". Om din post endast innehåller metadata kan du markera rutan "Metadata-only record".

Välj resurstypen för din post.

Lägg till skapare.

Om du inte hittar ditt ORCID kan du lägga till ditt namn manuellt.

Välj synlighetsinställningarna för din post.

Klicka på Förhandsvisa

Du kan förhandsgranska din post innan du skickar den. Om du behöver göra ändringar, klicka på "Tillbaka" för att redigera.

Efter att ha förhandsgranskat och gjort eventuella nödvändiga ändringar, klicka på "Skicka in för granskning".

Genom att skicka in din post godkänner du våra användarvillkor.

klicka på "Skicka in granskning".

Under godkännandeprocessen för din post kan du välja att chatta med community manager/curator.

Efter att ha skickat in din post kan du fortfarande se och redigera den.
